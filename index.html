AR.js + A-Frame + GitHub Pages

Realidad aumentada educativa gratuita desde el celular

Publica una experiencia web estatica para que estudiantes escaneen un codigo QR, apunten la camara a una imagen marcador y vean un modelo 3D sobre ella.

Crear experiencia Ver realidad aumentada Generar QR
1

Docentes

Elige una imagen facil de reconocer, convierte esa imagen a marcador NFT y sube los tres archivos generados al repositorio junto con el modelo GLB.

2

Estudiantes

Abre el enlace o escanea el QR, permite el uso de la camara y apunta hacia la imagen marcador impresa o mostrada en otra pantalla.

3

Publicacion

GitHub Pages es gratuito, pero no guarda archivos subidos desde la pagina. Para cambiar el modelo o marcador, debes reemplazar los archivos dentro del repositorio.

Codigo QR

Genera un QR para abrir el visor

Si ya publicaste el sitio en GitHub Pages, escribe la URL publica de viewer.html. Si estas probando localmente, usa el boton para completar la URL actual automaticamente.

El QR apuntara directamente a viewer.html.

Proyecto estatico listo para GitHub Pages. Sin backend, sin bases de datos y sin servicios pagos.
