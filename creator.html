Pasos obligatorios

Sube los archivos manualmente

  1. Coloca el modelo 3D en assets/models/modelo.glb.
  2. Coloca la imagen marcador en assets/images/marcador.jpg.
  3. Convierte la imagen marcador con NFT Marker Creator.
  4. Descarga marcador.fset, marcador.fset3 y marcador.iset.
  5. Sube los tres archivos a assets/markers/.
  6. Publica el repositorio con GitHub Pages y abre viewer.html.
